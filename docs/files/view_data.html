    view_data.php is a web page that allows easy viewing of mysql database auction data.

    This page provides both the frontend and backend. The front end comes from resources/html/view_data.html and resources/js/view_data.js. plotly and jquery are used to show graph the data. Requests are sent from the front end to this php for data. The mysql query for data over time has been highly optimized for speed. The raw data is sent back to the front end and processed (i.e. filtered) using javascript before graphing.
